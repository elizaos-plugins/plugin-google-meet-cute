This guide will help you set up OBS Studio as a virtual camera for Google Meet, allowing you to:

  • Use custom backgrounds and overlays
  • Apply filters and effects
  • Share your screen with annotations
  • Create professional presentations
  • Capture high-quality recordings

üìã Prerequisites

Step 1: Install OBS Studio

Download and install OBS Studio from the official website:

https://obsproject.com/download

Choose the version for your operating system (Windows, macOS, or Linux).

macOS Users: After installation, you'll need to grant OBS permission to access your camera and microphone in System Preferences ‚Üí Security & Privacy.

üîß OBS Configuration

Step 2: Basic OBS Setup

  1. Open OBS Studio
  2. Run the Auto-Configuration Wizard (Tools ‚Üí Auto-Configuration Wizard)
  3. Choose "Optimize just for recording, I will not be streaming"
  4. Set Base Resolution to 1920x1080 (or your preferred resolution)
  5. Set FPS to 30

Step 3: Create Your Scene

  1. In the Scenes panel, click the + button to create a new scene
  2. Name it "Google Meet"
  3. Add sources:
    • Video Capture Device: Your webcam
    • Audio Input Capture: Your microphone
    • Browser Source: For overlays or web content
    • Image: For backgrounds or logos

üéÆ Virtual Camera Setup

Step 4: Enable Virtual Camera

Windows/Linux:

  1. Click "Start Virtual Camera" in the Controls panel
  2. The button should change to "Stop Virtual Camera"

macOS:

  1. You may need to install the macOS Virtual Camera plugin separately
  2. Download from: OBS Virtual Camera Plugin
  3. After installation, restart OBS
  4. Click "Start Virtual Camera"
Tip: You can set OBS to automatically start the virtual camera on launch: Settings ‚Üí General ‚Üí "Start Virtual Camera" when OBS starts.

üåê Google Meet Configuration

Step 5: Select OBS Virtual Camera in Google Meet

  1. Join or start a Google Meet
  2. Click the three dots menu (‚ãÆ) in the bottom toolbar
  3. Go to Settings ‚Üí Video
  4. In the Camera dropdown, select:
    • Windows: "OBS Virtual Camera"
    • macOS: "OBS Virtual Camera"
    • Linux: "OBS Camera" or "v4l2loopback"
  5. You should see your OBS scene in the preview

üé® Advanced Features

Virtual Backgrounds

  1. Add a "Chroma Key" filter to your webcam source
  2. Use a green screen or solid color background
  3. Add an image or video source behind your webcam
  4. Adjust the chroma key settings for best results

Screen Sharing with Webcam

  1. Add a "Display Capture" source
  2. Position your webcam in a corner
  3. Add a border or frame around your webcam
  4. Use this scene when presenting

Audio Processing

  1. Right-click your Audio Input source
  2. Select "Filters"
  3. Add filters:
    • Noise Suppression: Remove background noise
    • Compressor: Even out volume levels
    • Gain: Boost quiet audio

üé¨ Recording Setup

Configure Recording Settings

  1. Go to Settings ‚Üí Output ‚Üí Recording
  2. Set Recording Path to your desired folder
  3. Recording Format: MP4
  4. Encoder: Hardware (NVENC/AMD/QuickSync) if available
  5. Recording Quality: "High Quality, Medium File Size"
  6. Click "Start Recording" to record your meeting

ü§ù ElizaOS Integration

Using with ElizaOS Google Meet Extension

The ElizaOS extension works seamlessly with OBS:

  1. Set up OBS as described above
  2. Start OBS Virtual Camera
  3. Select OBS Virtual Camera in Google Meet
  4. The extension will handle meeting automation while OBS handles video
  5. Audio is captured both by OBS (for recording) and the extension (for transcription)
Pro Tip: Create multiple scenes in OBS for different meeting scenarios:
  • "Casual Meeting" - Just webcam
  • "Presentation" - Screen share with webcam
  • "Professional" - Webcam with branded background
  • "Interview" - Split screen layout

üö® Troubleshooting

Common Issues:

  • Virtual Camera not showing: Restart OBS and Chrome
  • Black screen: Check that a scene is selected and has sources
  • No audio: Audio goes through your regular mic, not OBS (unless using Audio Monitor)
  • Performance issues: Lower the resolution or FPS in OBS settings
